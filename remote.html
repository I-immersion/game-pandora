📖 Pandora 2026 — mode d'emploi
Pour l'équipe accueil

Démarrer et faire tourner une partie

1. Lancer Pandora

  1. Double-clic sur Lancer Pandora.command sur le bureau du Mac
  2. Une fenêtre noire (le jeu) s'ouvre en plein écran + un terminal en arrière-plan
  3. Ouvrir la télécommande : c'est cette page actuelle, sur l'iPad ou un PC
  4. Vérifier en haut à droite : pastille verte = Connecté
Pastille rouge ? Toucher la pastille puis cliquer "Reconnecter". Sinon recharger la page (tirer vers le bas sur tablette, ou Cmd+Shift+R sur Mac).

2. Configurer la partie

  1. Mode : Timer (durée fixe) / Course (premier à X points) / Mort subite (jusqu'à la 1ère égalité battue)
  2. Joueurs : 2 à 6, mettre les noms si possible
  3. Durée / Objectif selon le mode
  4. Toucher DÉMARRER au moment voulu

3. Pendant la partie

  • Les buzzers physiques ajoutent les points automatiquement
  • Faux positif ? Carte du joueur → bouton ↶ Annuler
  • Pour ajouter un point à la main : bouton + sur la carte
  • STOP arrête la partie (gagnant désigné), RESET remet à zéro

4. Implication des joueurs (téléphones)

  1. Toucher 📱 QR codes → les codes s'affichent en plein écran sur la projection
  2. Chaque joueur scanne son QR et voit son score en direct sur son téléphone
  3. Les joueurs peuvent envoyer des messages flash via 4 boutons (🏎️ GO, ⚠️ VIRAGE, ⚡ BOOST, 🎉 BRAVO) → s'affichent en grand sur la projection

5. Messages depuis l'accueil

Champ Flash message en bas de la remote → taper un texte (ex: "Pause technique 5 min") → Envoyer. Affiché 5 secondes en orange clignotant sur la projection ET les téléphones joueurs.

6. LED des buzzers

Les LED RGB sont pilotées automatiquement :

  • Couleurs natives par joueur (orange, jaune, vert, cyan, violet, rouge)
  • Flash blanc bref quand un joueur marque
  • La LED du gagnant pulse à la fin

Bouton 💡 LED pour changer manuellement les couleurs ou tout éteindre.

7. En cas de souci

  • Pandora figé / planté → fermer la fenêtre + le terminal, relancer le .command
  • Télécommande "Déconnecté" → recharger la page (Cmd+Shift+R)
  • Buzzers ne réagissent plus → vérifier l'alim des Arduino + appeler JB
  • Téléphones joueurs ne se mettent pas à jour → vérifier le WiFi du Mac et des téléphones

8. Fin de soirée

  1. Fermer la fenêtre Pandora (clic rouge en haut à gauche)
  2. Dans le terminal : Cmd+W pour fermer
  3. Couper les buzzers Arduino
Pour JB (technique)

Architecture et protocoles

Stack

  • Electron 28+, node-osc 2.1, ws 8.16, qrcode 1.5.3
  • Vanilla JS/HTML/CSS, pas de framework, pas de build
  • 3 fichiers source : main.js (Node), index.html (projection), remote.html (télécommande)

Ports réseau

  • OSC IN : UDP 3333 — réception des buzzers
  • OSC OUT : UDP 7000 par défaut (config remote → Régie OSC)
  • WebSocket : TCP 8080 — télécommande

OSC entrant (depuis Arduino/MaKey)

/Touch <player_int>     # ajoute 1 point au joueur N (1-6)

OSC sortant (vers Chataigne/QLab/Resolume)

# État de partie
/pandora/state <string>        # idle|playing|suddendeath|finished|stopped
/pandora/mode <string>         # timer|race|suddendeath
/pandora/players <int>         # 2..6
/pandora/duration <int>        # minutes (timer)
/pandora/target <int>          # points (race)

# Triggers de jeu
/pandora/game/started
/pandora/game/stopped
/pandora/game/finished
/pandora/game/reset
/pandora/suddendeath/enter

# Scores
/pandora/score/<N>/value <int>
/pandora/score/<N>/added       # trigger à chaque point
/pandora/score/<N>/removed     # trigger annulation
/pandora/leader <int>          # N du leader
/pandora/winner/<N>            # trigger pour chaque gagnant
/pandora/winner/count <int>

# Messages
/pandora/message <string>
/pandora/message/clear

# LED RGB Arduino (v1.5)
/setColor <player> <r> <g> <b>  # tous int32, RGB 0-255

Effets LED auto (v1.6)

  • Flash point : /setColor N 255 255 255 pendant 150ms puis restore
  • Pulse gagnant : 6 cycles × 250ms (couleur ↔ 30% intensité)
  • Blink mort subite : joueurs à égalité clignotent à 400ms (couleur ↔ éteint)
  • Override manuel via panel 💡 LED est préservé : les effets restaurent la couleur courante de référence (pas forcément la native)

Firebase RTDB (page joueur)

Projet lumiia-live, région europe-west1

  • /pandora/live : état complet pushé toutes 2s pendant timer
    { state, mode, players, duration, target, total, remaining, names[], scores[], flash, updatedAt }
  • /pandora/inbox/<auto> : messages joueurs (text, from, at). main.js poll toutes 800ms et flush.

Page joueur

  • URL fixe : https://i-immersion.github.io/pandora-player/?p=N
  • Repo séparé : I-immersion/pandora-player (GitHub Pages)
  • Firebase Web SDK v10.12 modular, lecture realtime via onValue
  • 4 boutons messages → push dans /pandora/inbox

Utile pour vérifier ce qui part vraiment côté Mac quand Chataigne ne réagit pas.
